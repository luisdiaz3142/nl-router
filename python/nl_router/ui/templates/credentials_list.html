Credentials

AES-256-GCM-encrypted secrets referenced by destinations.

The UI shows only metadata — name, kind, last-used, expires-at. Plaintext payloads are never returned to any client; they live encrypted in the database and are decrypted in-process by the dispatcher. Create and rotate via the CLI: nl-router credential create, nl-router credential rotate. A per-kind UI form is on the follow-up list.
